MNB SOAP Kliens

Ez a modul a Magyar Nemzeti Bank hivatalos webszolgáltatásával (ArfolyamService) létesít biztonságos kapcsolatot. A megoldás demonstrálja a klasszikus nagyvállalati SOAP protokoll használatát Java környezetben.

Implementált funkciók:

  • WSDL alapú kódgenerálás: A kliens oldali Java osztályokat a Maven JAX-WS pluginja generálja automatikusan a szolgáltatásleíró alapján.
  • XML Feldolgozás: A szerver válaszaként érkező nyers XML adatfolyamot DOM Parser segítségével dolgozzuk fel és alakítjuk Java objektumokká.
  • Adatvizualizáció: A kinyert árfolyamadatokat a frontend oldalon a Chart.js könyvtár segítségével interaktív vonaldiagramon ábrázoljuk.
  • Árfolyamok lekérdezése

Oanda Forex Kereskedés

Valós idejű devizakereskedési szimuláció az Oanda v20 REST API segítségével. A modul teljes körű hozzáférést biztosít a kereskedési fiókhoz, beleértve a számlaadatokat, árakat és pozíciók kezelését.
